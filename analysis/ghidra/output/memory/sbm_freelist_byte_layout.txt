PART 2: SBM POOL OBJECT LAYOUT
======================================================================

From FUN_00aa6aa0 (alloc) and FUN_00aa6c70 (free):
  pool + 0x04: base address of arena
  pool + 0x08: freelist head pointer
  pool + 0x20: spinlock
  pool + 0x24: lock counter / reentrant depth
  pool + 0x40: block size
  pool + 0x48: per-page allocation count array
  pool + 0x50: total arena size
  pool + 0x58: total allocated page count

The freelist is a singly-linked list. Each freed block stores
the next pointer somewhere in its body. FUN_00aa6e00 reveals WHERE.
